# Szablon-LTR

Ten katalog zawiera szablony artefaktow dla modelu Literate Theoretical Research (LTR).

Zasady ogolne:
- Tryb pracy ustawiasz w metadanych (formalny/badacz/hybrydowy).
- Przelaczanie w tekscie: uzyj tagu sekcji, np. [TRYB: FORMALNY] lub [TRYB: BADACZ].
- Dla sekcji formalnych stosuj kroki, oznaczenia i status walidacji (CAS/reczna).
- Powiazane artefakty zapisuj w formacie [ID:Typ], np. [LTR-DERIV-001:Raport].
- Taguj rownania w raportach: po wzorze dodaj w osobnej linii tag EQ:ID.
- W Mapie Notacji wpisz globalne zalozenia jednostkowe (np. c=1, hbar=1).
