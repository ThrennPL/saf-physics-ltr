# Szablon-Fizyka

Szablony dokumentow badawczych dla pracy w SAF.

Zasady:
- Wypelniaj sekcje, nawet jesli wstepnie zawieraja szkic.
- Decyzje oznaczaj ID i powiazuj z artefaktami.
- Wnioski oddzielaj od faktow.
- Status roboczy: OK / Warning / Blocker (jesli dotyczy).
- Pewnosc: liczba 0-1 (jesli dotyczy).
- Pytania eskalacyjne: Q-XXX z priorytetem niski/sredni/wysoki (jesli dotyczy).
